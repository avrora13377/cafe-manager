# Cafe Manager — Управление заказами кафе

Десктопное приложение для автоматизации учета заказов в предприятиях общественного питания.

Разработано в рамках производственной практики (по профилю специальности)  
ПМ.01 «Разработка модулей программного обеспечения для компьютерных систем»  
Специальность: 09.02.07 Информационные системы и программирование

## Функциональность

- **Меню** — ведение справочников категорий и блюд
- **Заказы** — создание заказов, управление статусами (Принят → Готовится → Готов → Оплачен), добавление/удаление позиций
- **Отчеты** — формирование статистики продаж за период (по дням и по блюдам)

## Технологии

- Python 3.11
- tkinter / ttk
- SQLite
- Git / GitHub

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/avrora13377/cafe-manager.git
cd cafe-manager

# Заполнить тестовыми данными (опционально)
python seed_data.py

# Запустить приложение
python main.py
```

## Статусы заказов

`Принят → Готовится → Готов → Оплачен`

Отмена возможна на этапах «Принят» и «Готовится».
